另类栈
时间限制：C/C++语言 1000MS；其他语言 3000MS
内存限制：C/C++语言 65536KB；其他语言 589824KB
题目描述：
传统的栈一般只支持三种操作，即插入栈顶元素，弹出栈顶元素，和获取栈顶元素值。

现在有一种新型栈，支持4种操作：

1. push v 表示把v元素压入栈中。

2. pop 表示弹出当前的栈顶元素，如果栈空则无操作。

3. delete v 表示删除距离栈顶最近的v元素，如果不存在v，则不删除。

4. query 询问栈顶元素的大小，如果栈为空，输出0。

现在给你n个操作，对于每一个query操作，输出栈顶元素的大小。 

输入
输入第一行是一个整数n（1≤n≤100000）,表示操作的数量。

接下来有n行，每行描述一个操作，如问题描述所示,其中v是一个1到5之间的正整

输出
对于每一个query，输出一行，即栈顶元素的大小。


样例输入
7
push 3
push 1
push 2
delete 1
query
pop
query
样例输出
2
3

Hint
首先插入3个元素，得到{3，1，2}，然后距离栈顶最近的1，得到{3，2},query得到2，然后pop，弹出2，query输出3。
